# ImpactPlay 🏆

A high-performance SaaS + Lottery platform that connects tournament performance with charitable giving.

## 🚀 Deployed Link
**[Live Demo](https://impact-play-digital-heroes.vercel.app)** 

## 🎯 Evaluator Quick Start (Test in 1 Minute)
To save you time, the login page contains **Demo Cards**. Simply click them to auto-fill credentials:

1. **Admin Portal**: Click the blue "Demo Admin" card to access the Draw System.
2. **User Dashboard**: Click the violet "Demo User" card to view scores and winnings.

### Key Features Under the Hood:
- **Stableford Logic**: Strict 1-45 point range enforced on the backend.
- **Dynamic Pruning**: Database trigger automatically retains only the top 5 latest scores.
- **Stripe Integration**: Full simulated PCI-compliant checkout flow with "Test Mode" card support.
- **Draw Engine**: Admin-controlled prize distribution and charity splitting logic.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Database**: Supabase / PostgreSQL
- **ORM**: Prisma 6 (Transaction Mode)
- **Styling**: Tailwind CSS / Shadcn UI
- **Auth**: Supabase Auth
- **Payments**: Stripe (Pre-integrated Demo)
